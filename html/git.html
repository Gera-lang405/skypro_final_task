Git

  • git init - инициализирует новый локальный репозиторий Git в текущей папке.
  • git clone - клонирует удалённый репозиторий на локальный компьютер.
  • git add - добавляет файлы в индекс (область подготовки) для последующего коммита.
  • git commit - сохраняет подготовленные изменения в истории репозитория.
  • git status - показывает текущее состояние рабочей директории и индекса.
  • git push - отправляет локальные коммиты в удалённый репозиторий.
  • git pull - загружает и интегрирует изменения из удалённого репозитория в текущую ветку.
  • git branch - список веток, создание, переименование и удаление веток.
